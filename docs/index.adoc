== Available Skills

[cols="1,3"]
|===
|link:skills/verify-content.html[*`/verify-content`*]
|Validate workshop/demo content against Red Hat Style Guide, AsciiDoc standards, and WCAG 2.1 AA accessibility requirements.

|link:skills/create-lab.html[*`/create-lab`*]
|Create workshop modules with sequential questioning, auto-generated structure, and verification checkpoints.

|link:skills/create-demo.html[*`/create-demo`*]
|Create presenter demos with Know/Show methodology, business impact focus, and competitive positioning.

|*`/generate-agv-description`*
|Generate AgnosticV catalog `description.adoc` files for RHDP provisioning. Auto-extracts content from Showroom guides.

|*`/blog-generate`*
|Transform workshop/demo content into blog post format for marketing and developer content.
|===
